# Como adicionar o vídeo da especialista

1. Grave/exporte o vídeo em formato .mp4 (recomendo até 15-20MB, resolução 720p ou 1080p, proporção vertical ou 4:5 pra combinar com o formato da moldura no hero).
2. Renomeie o arquivo para: `video-especialista.mp4`
3. Coloque ele dentro desta pasta `assets/` no repositório (mesmo nível deste README).
4. Faça o commit/push. O site já está configurado para carregar automaticamente de `assets/video-especialista.mp4`.

A foto que já está no site continua servindo de capa (poster) do vídeo até ele carregar/tocar.
